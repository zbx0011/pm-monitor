# 铂钯套利监控系统 - 重启指南

## 1. 本地电脑 (Windows) 重启后

**只需要做一步：**
1. 双击运行文件夹中的 **`启动监控.bat`**

> **系统会自动执行以下操作：**
> - 启动 API 服务器
> - 启动 10分钟/次 的自动数据采集
> - 自动打开浏览器监控页面

---

## 2. 远程服务器 (VPS) 重启后

如果 VPS 重启了，请 SSH 连接到服务器，然后复制并执行以下**一行命令**即可恢复所有服务：

```bash
cd ~/monitor && git pull && pkill -f price_api_server.py; sleep 1; nohup python3 price_api_server.py > api.log 2>&1 & sleep 3 && echo "服务已启动"
```

> **说明：**
> - 这条命令会自动拉取最新代码
> - 停止旧服务并启动新服务
> - 启动后，VPS 也会每 10 分钟自动拉取 GitHub 上的最新数据

---

## 3. 自动更新机制说明

- **本地电脑**：作为数据源头。它会每10分钟运行一次采集脚本，获取广期所和CME的最新数据，并自动上传到 GitHub。
- **VPS服务器**：作为展示端。它会每10分钟自动从 GitHub 拉取最新的数据文件。

**只要保持本地电脑的黑色命令行窗口（bat运行的窗口）开启，数据就会持续更新。**
